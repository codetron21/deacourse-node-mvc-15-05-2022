# MOBILE LEGEND MVC APP
Submission project from Deacourse, YT channel [DeaAfrizal](https://www.youtube.com/c/DeaAfrizal).

## Tech Stack
- Node JS.
- Express (backend).
- EJS (front-end).

## Architecture
- Using MVC pattern.
  - Model
    - Hero.
    - History.
  - Controller
    - Hero.
    - History.
  - View
    - index (list page).
    - detail page.
    - history page.
